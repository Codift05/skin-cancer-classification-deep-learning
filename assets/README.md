# Assets Folder

## How to Add Team Photo

1. **Save your team photo** (the medical doctor image) as `team_photo.jpg` in this folder
2. The image should be in JPG or PNG format
3. Recommended size: 1200x600 pixels or similar aspect ratio

## Current Path
```
assets/team_photo.jpg
```

The app will automatically load this image as the background for the Team Photo section.
